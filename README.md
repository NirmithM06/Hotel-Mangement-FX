# 🏨 Hotel Management System (JavaFX + Maven)

## 📌 Project Description

This is a **Hotel Management System** developed using **JavaFX** and **Maven**.
It allows users to manage hotel rooms, bookings, and customer details efficiently.

---

## 🚀 Features

* Add new rooms 🛏️
* View available rooms 👀
* Book rooms 📅
* Checkout functionality ✅
* Simple UI using JavaFX 🎨
* Database integration (SQLite) 💾

---

## 🛠️ Technologies Used

* Java
* JavaFX
* Maven
* SQLite
